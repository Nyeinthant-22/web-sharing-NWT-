Day 22: List (Array) တွေကို လှည့်ပတ်ဖို့ နောက်ထပ်နည်းလမ်း - while Loop

မင်္ဂလာပါ! 👋 မနေ့က List (Array) တွေကို for...of loop နဲ့ ဘယ်လို လှည့်ပတ်ကြည့်ရမလဲဆိုတာ လေ့လာခဲ့ကြတယ်။ ဒီနေ့မှာတော့ Array တွေကို လှည့်ပတ်ဖို့ နောက်ထပ် အရေးကြီးတဲ့ Loop အမျိုးအစားတစ်ခုဖြစ်တဲ့ while loop အကြောင်း လေ့လာသွားကြမယ် 😉။

while Loop ဆိုတာ ဘာလဲ?

while loop ကတော့ သတ်မှတ်ထားတဲ့ အခြေအနေတစ်ခု မှန်နေသရွေ့ (true ဖြစ်နေသရွေ့) Code တွေကို ထပ်ခါထပ်ခါ လုပ်ဆောင်ပေးပါတယ်။ `for` loop လိုမျိုး အကြိမ်ရေ အတိအကျ မသိတဲ့အခါမျိုးမှာ သုံးဖို့ ပိုအဆင်ပြေပါတယ်။

while Loop က ဘယ်လို အလုပ်လုပ်လဲ?

  1. while loop စတဲ့အခါမှာ ပေးထားတဲ့ အခြေအနေကို စစ်ဆေးပါတယ်။
  2. အခြေအနေက true ဆိုရင် Loop ထဲက Code တွေကို လုပ်ဆောင်ပါတယ်။
  3. Code တွေ လုပ်ဆောင်ပြီးရင် အခြေအနေကို ပြန်စစ်ပါတယ်။
  4. အခြေအနေက true ဖြစ်နေသေးရင် Code တွေကို ထပ်လုပ်ဆောင်ပါတယ်။ false ဖြစ်သွားမှ Loop က ရပ်သွားမှာပါ။

ရိုးရှင်းသော Code ဥပမာ (JavaScript ဖြင့်):

let colors = ["အနီ", "အဝါ", "အစိမ်း", "အပြာ"];
let i = 0; // စမှတ် (index 0 ကနေ စမယ်)

while (i < colors.length) { // i က Array ရဲ့ အရှည်ထက် ငယ်နေသရွေ့ Loop လည်ပတ်မယ်
  console.log(colors[i]); // လက်ရှိ index မှာရှိတဲ့ အရောင်ကို ထုတ်ပြမယ်
  i++; // index ကို ၁ တိုးပေးမယ် (နောက်တစ်ရောင်ကို ရည်ညွှန်းဖို့)
}

Code ရဲ့ ရှင်းလင်းချက်:

  • let colors = ["အနီ", "အဝါ", "အစိမ်း", "အပြာ"];: ဒါက colors ဆိုတဲ့ Array တစ်ခုကို ဖန်တီးလိုက်တာပါ။
  • let i = 0;: i ဆိုတဲ့ Variable ကို 0 နဲ့ စတင်သတ်မှတ်လိုက်ပါတယ်။ i က Array ရဲ့ index ကို ကိုယ်စားပြုပါတယ်။
  • while (i < colors.length): ဒါက while loop ရဲ့ အခြေအနေပါ။ i ရဲ့ တန်ဖိုးဟာ colors Array ရဲ့ အရှည်ထက် ငယ်နေသရွေ့ Loop က ဆက်လည်ပတ်နေပါမယ်။
  • console.log(colors[i]);: ဒါက Loop တစ်ခါလည်တိုင်း လုပ်ဆောင်မယ့် Code ပါ။ colors[i] က လက်ရှိ i ရဲ့ တန်ဖိုး (index) မှာရှိတဲ့ အရောင်ကို ထုတ်ပြပါတယ်။
  • i++;: Loop တစ်ခါလည်ပြီးတိုင်း i ရဲ့ တန်ဖိုးကို ၁ တိုးပေးပါတယ်။ ဒါမှ နောက်တစ်ကြိမ် လည်တဲ့အခါ နောက်ထပ် အရောင်ကို ထုတ်ပြမှာပါ။

while Loop ကို ဘယ်အချိန်မှာ သုံးမလဲ?

  • Loop ဘယ်နှစ်ခါ လည်ရမယ်ဆိုတာ အတိအကျ မသိတဲ့အခါ (ဥပမာ: User က input ထည့်နေသရွေ့)
  • အခြေအနေတစ်ခု ပြည့်စုံတဲ့အထိ Code တွေကို ထပ်ခါထပ်ခါ လုပ်ဆောင်ချင်တဲ့အခါ

နောက်ထပ် ဥပမာ

let numbers = [1, 2, 3, 4, 5];
let total = 0;
let j = 0;

while (j < numbers.length) {
  total += numbers[j]; // Array ထဲက နံပါတ်တွေကို ပေါင်းထည့်
  j++;
}

console.log("စုစုပေါင်း: " + total); // စုစုပေါင်းကို ထုတ်ပြမယ် (15)

မှတ်သားရန်:

while loop ကို သုံးတဲ့အခါ Loop က မဆုံးတော့ဘဲ ထာဝရလည်နေနိုင်တဲ့ အခြေအနေမျိုးကို သတိထားပါ။ အဲ့ဒါကို infinite loop လို့ခေါ်ပါတယ်။ Loop ရပ်မယ့် အခြေအနေကို သေချာ သတ်မှတ်ထားဖို့ လိုပါတယ်။

ဒီနေ့မှာတော့ List (Array) တွေကို လှည့်ပတ်ဖို့ နောက်ထပ် နည်းလမ်းတစ်ခုဖြစ်တဲ့ while loop အကြောင်း လေ့လာခဲ့ကြပါတယ်။ while loop ဟာ Programming မှာ အရေးပါတဲ့ Loop အမျိုးအစားတစ်ခုဖြစ်ပြီး အခြေအနေပေါ်မူတည်ပြီး Code တွေကို ထပ်ခါထပ်ခါ လုပ်ဆောင်ချင်တဲ့အခါ အသုံးဝင်ပါတယ်။ နောက်ရက်တွေမှာတော့ Loop တွေနဲ့ ပတ်သက်တဲ့ နောက်ထပ် အသုံးဝင်တဲ့ အကြောင်းအရာတွေကို ဆက်ပြီး လေ့လာသွားကြမယ်နော်။ 😊
