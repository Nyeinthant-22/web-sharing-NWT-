Day 18: List (Array) ထဲကို အချက်အလက်တွေ ထပ်ထည့်ခြင်း

မင်္ဂလာပါ! 👋 မနေ့က Array ထဲက တန်ဖိုးတွေကို ပြောင်းလဲနည်းကို လေ့လာခဲ့ကြပါတယ်။ ဒီနေ့မှာတော့ Array ထဲကို အသစ်ထည့်တဲ့နည်းလမ်းတွေကို လေ့လာသွားကြမယ် 😉။

1. push() နည်းလမ်း

push() ကို သုံးတဲ့အခါ Array ရဲ့ နောက်ဆုံးမှာ တန်ဖိုးအသစ်ထပ်ထည့်တယ်။ 

let colors = ["red", "green"];
colors.push("blue");
console.log(colors); // ["red", "green", "blue"]

2. unshift() နည်းလမ်း

unshift() ကို သုံးတဲ့အခါ Array ရဲ့အစမှာ တန်ဖိုးအသစ်ထပ်ထည့်တယ်။ 

let fruits = ["banana", "orange"];
fruits.unshift("apple");
console.log(fruits); // ["apple", "banana", "orange"]

ရှင်းလင်းချက်

  • push() ဟာ နောက်ဆုံး Index မှာ ထပ်ထည့်တာပါ။
  • unshift() က အစဆုံး Index မှာ ထပ်ထည့်တာပါ။

ဒီနေ့ကနေပြီး Array ထဲကို တန်ဖိုးအသစ်တွေ ထည့်နည်းကိုသိပြီးဖြစ်ပြီ။ နောက်အကြောင်းအရာတွေမှာတော့ Element တွေကို ဖျက်တာ (remove) နည်းကို လေ့လာသွားကြမယ်နော်။ 😊
