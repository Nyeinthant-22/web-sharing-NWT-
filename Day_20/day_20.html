Day 20: List (Array) ထဲက အချက်အလက်တွေကို တစ်ခုချင်းစီ လှည့်ပတ်ကြည့်ခြင်း

မင်္ဂလာပါ! 👋 မနေ့က Array ထဲက အချက်အလက်တွေကို ဘယ်လို ဖယ်ရှားရမလဲဆိုတာ လေ့လာခဲ့ကြတယ်နော်။ ဒီနေ့မှာတော့ Array ထဲမှာရှိတဲ့ အချက်အလက်တွေကို တစ်ခုချင်းစီ လှည့်ကြည့်ဖို့အတွက် Loop နဲ့ ဘယ်လိုလုပ်ရမလဲဆိုတာ လေ့လာကြမယ်။

1. Looping ဆိုတာဘာလဲ?

Looping ဆိုတာက Array ထဲမှာရှိတဲ့ တန်ဖိုးတွေကို တစ်ခုချင်းစီ အလှည့်ကျလုပ်ဆောင်ဖို့အသုံးပြုတဲ့နည်းလမ်းတစ်ခုပါ။ ဥပမာအားဖြင့် မင်းမှာ fruits အမည်တွေထည့်ထားတဲ့ array ရှိတယ်ဆိုပါစို့၊ မင်းက `console.log()` နဲ့ တစ်ခုချင်းစီကိုဖော်ပြချင်တယ်။ အဲ့ဒီအချိန်မှာ Loop ကအထောက်အကူဖြစ်ပါတယ်။

2. `for` loop နဲ့ Looping

JavaScript မှာ `for` loop ကိုအသုံးပြုပြီး Array ထဲက element တစ်ခုချင်းစီကို ပြသလို့ရတယ်။ 

let myFruits = ["apple", "banana", "orange", "grape"];

for (let i = 0; i < myFruits.length; i++) {
  console.log(myFruits[i]);
}

ရှင်းလင်းချက်

  • i = 0 က Loop ကို 0 (ပထမ element) ကစတင်တာပါ။
  • i < myFruits.length ဆိုတာက array ရဲ့ နောက်ဆုံးထိသာ loop run ဖို့ဖြစ်တယ်။
  • myFruits[i] က index အလိုက် array ထဲက တန်ဖိုးကိုဖော်ပြတာ။

3. တခြား Loop ပုံစံ - `for...of`

ES6 မှာအသစ်ဖြစ်တဲ့ `for...of` loop ကိုလည်း Array အတွက် အသုံးပြုလို့ရတယ်။ 

for (let fruit of myFruits) {
  console.log(fruit);
}

ရှင်းလင်းချက်

  • fruit ဟာ Array ထဲက တစ်ခုချင်းစီကို ကိုယ်စားပြုတယ်။
  • Code ပိုပြီး ဖတ်ရလွယ်တယ်။

နောက်တစ်နေ့မှာတော့ Loop ကို အသုံးပြုပြီး Array ထဲက တန်ဖိုးတွေကို ပြောင်းလဲဖို့နည်းလမ်းတွေကို လေ့လာသွားမယ်။ 😉
