Day 24: List (Array) ထဲက သတ်မှတ်ထားတဲ့ အချက်အလက်တွေကို ရှာဖွေခြင်း (Finding Specific Elements in a List/Array)

မင်္ဂလာပါ! 👋 လွန်ခဲ့တဲ့ရက်တွေက Loop တွေကို သုံးပြီး Array ထဲက ဂဏန်းတွေကို ဘယ်လို ပေါင်းရမလဲဆိုတာ လေ့လာခဲ့ကြတယ်နော်။ ဒီနေ့မှာတော့ Loop တွေရဲ့ နောက်ထပ် အသုံးဝင်တဲ့ အသုံးချမှုတစ်ခုကို လေ့လာသွားကြမယ်။ အဲ့ဒါကတော့ Array ထဲက သတ်မှတ်ထားတဲ့ အချက်အလက် (သို့မဟုတ်) အခြေအနေတစ်ခုနဲ့ ကိုက်ညီတဲ့ အချက်အလက်တွေကို ရှာဖွေခြင်း ပါပဲ 😉။

ဘာကြောင့် Array ထဲက အချက်အလက်တွေကို ရှာဖွေဖို့ လိုအပ်တာလဲ?

စဉ်းစားကြည့်ပါ... မင်းမှာ ကျောင်းသားနာမည်စာရင်း ၁၀၀ ရှိတယ် 🧑‍🤝‍🧑။ အဲ့ဒီထဲက "မောင်မောင်" ဆိုတဲ့ နာမည်ပါတဲ့ ကျောင်းသားကို ရှာချင်တယ်ဆိုပါစို့။ ဒါမှမဟုတ် အမှတ် ၅၀ အထက်ရတဲ့ ကျောင်းသားတွေရဲ့ နာမည်စာရင်းကို ထုတ်ချင်တယ်ဆိုပါစို့။ ဒီလိုမျိုး သတ်မှတ်ချက်တွေနဲ့ ကိုက်ညီတဲ့ အချက်အလက်တွေကို ရှာဖွေတဲ့အခါ Loop နဲ့ Conditional Statement (if) ကို ပေါင်းစပ်ပြီး သုံးလို့ရပါတယ်။

Loop နဲ့ if Statement ကိုသုံးပြီး အချက်အလက် ရှာဖွေနည်း

ဒီလုပ်ဆောင်ချက်အတွက် for...of loop က အရမ်းအဆင်ပြေပါတယ်။ Array ထဲက element တစ်ခုချင်းစီကို စစ်ဆေးဖို့ for...of loop ကို သုံးပြီး၊ အဲ့ဒီ element က သတ်မှတ်ထားတဲ့ အခြေအနေနဲ့ ကိုက်ညီလားဆိုတာ စစ်ဆေးဖို့ if statement ကို သုံးပါတယ်။

ရိုးရှင်းသော Code ဥပမာ (JavaScript ဖြင့်):

let fruits = ["apple", "banana", "orange", "grape", "apple", "kiwi"];
let fruitToFind = "apple"; // ရှာချင်တဲ့ အသီးနာမည်

console.log("ရှာဖွေနေသည်: " + fruitToFind);
console.log("--------------------");

// fruits Array ထဲက အသီးတစ်ခုချင်းစီကို လှည့်ပတ်ကြည့်မယ်
for (let fruit of fruits) {
  // လက်ရှိ 'fruit' က ရှာချင်တဲ့ 'fruitToFind' နဲ့ တူလား စစ်ဆေးမယ်
  if (fruit === fruitToFind) {
    console.log("တွေ့ပါပြီ! " + fruit + " ကို တွေ့ရှိသည်!");
  }
}
console.log("--------------------");

// နောက်ထပ် ဥပမာ: အမှတ် ၈၀ အထက်ရတဲ့ ကျောင်းသားအမှတ်တွေကို ရှာမယ်
let studentScores = [65, 82, 90, 70, 55, 88, 75];
console.log("\nအမှတ် ၈၀ အထက်ရရှိသူများ:");
console.log("--------------------");

for (let score of studentScores) {
  if (score >= 80) { // score က 80 ထက် ကြီးလား၊ 80 နဲ့ တူလား စစ်ဆေးမယ်
    console.log("ကောင်းတယ်! " + score + " မှတ် ရရှိသည်!");
  }
}
console.log("--------------------");

Code ရဲ့ ရှင်းလင်းချက်:

  • let fruits = ["apple", ...];: ဒါက အသီးနာမည်တွေပါတဲ့ fruits Array ကို ဖန်တီးလိုက်တာပါ။
  • let fruitToFind = "apple";: ဒါက ကျွန်တော်တို့ ရှာချင်တဲ့ အသီးနာမည်ကို fruitToFind ဆိုတဲ့ Variable ထဲမှာ ထည့်ထားတာပါ။
  • for (let fruit of fruits): fruits Array ထဲက အသီးနာမည်တစ်ခုချင်းစီကို လှည့်ပတ်ကြည့်ဖို့ for...of loop ကို စတင်လိုက်တာပါ။ Loop တစ်ပတ်စီမှာ fruit Variable ထဲကို လက်ရှိအသီးနာမည် ရောက်လာပါမယ်။
  • if (fruit === fruitToFind): ဒါက if statement ပါ။ လက်ရှိ fruit (ဥပမာ: "apple") ဟာ fruitToFind ("apple") နဲ့ === (တန်ဖိုးရော၊ အမျိုးအစားပါ) တူညီလားဆိုတာကို စစ်ဆေးပါတယ်။
    • တကယ်လို့ တူညီတယ် (အခြေအနေက true ဖြစ်တယ်) ဆိုရင် if block ထဲက console.log() ကို လုပ်ဆောင်ပါမယ်။
    • မတူညီဘူး (အခြေအနေက false ဖြစ်တယ်) ဆိုရင် if block ထဲက Code တွေကို ကျော်သွားပါမယ်။
  • ဒုတိယဥပမာမှာလည်း အလားတူပါပဲ။ studentScores ထဲက score တစ်ခုချင်းစီကို လှည့်ပတ်ပြီး score >= 80 ဆိုတဲ့ အခြေအနေနဲ့ ကိုက်ညီလားဆိုတာကို if statement နဲ့ စစ်ဆေးသွားတာပါ။

ဒီနေ့မှာတော့ Loop နဲ့ if statement ကို ပေါင်းစပ်ပြီး Array ထဲက သတ်မှတ်ထားတဲ့ အချက်အလက်တွေကို ဘယ်လို ရှာဖွေရမလဲဆိုတာကို လေ့လာခဲ့ကြပါတယ်။ ဒါဟာ Data တွေကို စီမံခန့်ခွဲရာမှာ အရေးကြီးတဲ့ အဆင့်တစ်ခုဖြစ်ပြီး နောက်ပိုင်းမှာ ပိုပြီးရှုပ်ထွေးတဲ့ Program တွေ ရေးတဲ့အခါ အများကြီး အသုံးဝင်မှာပါ။ ဒီနေ့သင်ခန်းစာကိုလည်း လက်တွေ့လေ့ကျင့်ကြည့်ဖို့ မမေ့နဲ့နော်! 😊
