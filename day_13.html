NWT Beginner Programming Concept

ဥပမာ ၁: မိုးရွာခြင်း 🌧️

အကယ်၍ မိုးရွာနေရင် ထီးယူသွားပါ။ မဟုတ်ရင်တော့ မယူသွားနဲ့။

if (raining == true) {
"Take Umbrella";
}

ဥပမာ ၂: ဗိုက်ဆာခြင်း 😋

အကယ်၍ ဗိုက်ဆာရင် ထမင်းစားပါ။

if (hungry == true){
"Eat Meal";
}

ဥပမာ ၃: အသက်စစ်ဆေးခြင်း 🎉

အကယ်၍ အသက် ၁၈ နှစ် ပြည့်ပြီးရင် မဲပေးနိုင်ပါတယ်။

if(age >= 18) {
"Can Vote";
}

ဥပမာ ၄: အပူချိန် 🥵

အကယ်၍ အပူချိန် ၃၀ ဒီဂရီထက် များရင် ရေကူးကန်သွားပါ။

if (temperature > 30) {
"Go to Swimming Pool";
}
