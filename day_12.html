Day 12: အခြေအနေပေါ် မူတည်တဲ့ လုပ်ဆောင်ချက် - If

မင်္ဂလာပါ! 👋 လွန်ခဲ့တဲ့ နှစ်ရက်က Comparison အကြောင်း လေ့လာပြီး ဖြစ်ပါတယ်။ ဒီနေ့တော့ အဲ့ဒီအခြေအနေပေါ် မူတည်ပြီး လုပ်ဆောင်ဖို့ **If** ကို သုံးမယ်နော်။

📌 "If" ဆိုတာ ဘာလဲ?

“If” က Programming မှာ "အကယ်၍" ဆိုတဲ့ အဓိပ္ပါယ်နဲ့ တူပါတယ်။ အခြေအနေတစ်ခု မှန်ရင် တစ်ခုခုလုပ်ဆောင်တယ်။ မမှန်ရင် ကျော်သွားတယ်။

🧾 ရိုးရှင်းတဲ့ ပုံစံ:

if (status == true) { "Continue Working This" } else { "Stop It" }

🌦️ နေ့စဉ်ဘဝနဲ့ ဥပမာ:

  • 🌧️ အကယ်၍ မိုးရွာရင် — ထီးယူပါ။
  • 😋 အကယ်၍ ဗိုက်ဆာရင် — ထမင်းစားပါ။
  • 🌡️ အကယ်၍ အပူချိန် ၃၀ ဒီဂရီ ကျော် — လေအေးပေးစက်ဖွင့်ပါ။

💻 Programming မှာ "If" ဘယ်လို အလုပ်လုပ်လဲ?

Programming မှာလည်း အခြေအနေ တစ်ခုကို စစ်ဆေးပြီး ဟုတ်ရင် code ကို လုပ်ဆောင်တယ်။ မဟုတ်ရင် ကျော်သွားတယ်။

🔜 နောက်သင်ခန်းစာ

နောက်တစ်နေ့မှာတော့ If statement ကို အသုံးပြုပြီး လုပ်ဆောင်ချက်တွေကို လုပ်နိုင်တဲ့ အကြောင်း ရှင်းပြသွားပါမယ်။
