Day 11: နှိုင်းယှဉ်ခြင်း လုပ်ဆောင်ချက်တွေ (Comparison Operators)

မင်္ဂလာပါ! 👋 မနေ့က Programming မှာ နံပါတ်တွေနဲ့ သင်္ချာတွက်ချက်မှုတွေ လုပ်ဖို့ သုံးတဲ့ သင်္ချာ လုပ်ဆောင်ချက်တွေအကြောင်း လေ့လာခဲ့ကြတယ်နော်။ ဒီနေ့မှာတော့ တန်ဖိုးနှစ်ခုကို နှိုင်းယှဉ်ကြည့်ဖို့ သုံးတဲ့ နှိုင်းယှဉ်ခြင်း လုပ်ဆောင်ချက်တွေ (Comparison Operators) အကြောင်း ပြောပြမယ်။

Programming မှာ တန်ဖိုးတွေကို နှိုင်းယှဉ်ဖို့ အောက်ပါ သင်္ကေတတွေကို အများဆုံး သုံးပါတယ် 👇

၁။ တူညီခြင်း (==)

5 == 5 → True

5 == 10 → False

၂။ မတူညီခြင်း (!=)

5 != 10 → True

5 != 5 → False

၃။ ပိုကြီးခြင်း (>)

10 > 5 → True

5 > 10 → False

၄။ ပိုငယ်ခြင်း (<)

3 < 7 → True

7 < 3 → False

၅။ ပိုကြီးခြင်း သို့မဟုတ် တူညီခြင်း (>=)

10 >= 5 → True

5 >= 5 → True

၆။ ပိုငယ်ခြင်း သို့မဟုတ် တူညီခြင်း (<=)

3 <= 7 → True

10 <= 5 → False

ဒီနေ့မှာတော့ အခြေခံ Comparison Operators (၆) ခုကို လေ့လာခဲ့ပြီး၊ နောက်တစ်နေ့မှာတော့ If Statement နဲ့ ဘယ်လို တွဲသုံးရမယ်ဆိုတာ ဆက်လက်လေ့လာသွားမယ်နော်! 😊
