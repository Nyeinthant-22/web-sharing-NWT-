List (Array) Index ဆိုတာ ဘာလဲ?

Array ထဲက တစ်ခုချင်းစီကို သူ့ရဲ့ Index နဲ့ ရယူနိုင်တယ်။ Index က 0 က စတယ်။ 

let myColors = ["အနီ", "အဝါ", "အစိမ်း", "အပြာ"];
//            [  0  ,   1  ,   2  ,   3  ]

Array ထဲက Data ကို Access လုပ်မယ်

ArrayName[index] နဲ့ ရယူနိုင်တယ်။

let myColors = ["အနီ", "အဝါ", "အစိမ်း", "အပြာ"];

console.log(myColors[0]); // "အနီ"
console.log(myColors[2]); // "အစိမ်း"

let firstColor = myColors[0];
console.log(firstColor);

ရှင်းလင်းချက်

  • myColors[0] ဆိုတာ index 0 မှာရှိတဲ့ data ကို ယူတာပါ။
  • let firstColor = myColors[0] ဆိုတာ "အနီ" ကို variable ထဲထည့်လိုက်တာပါ။

နောက်ထပ် ဥပမာ

let myNumbers = [10, 20, 30, 40, 50];

let thirdNumber = myNumbers[2];
console.log(thirdNumber + 5);

console.log(myNumbers[myNumbers.length - 1]);

ရှင်းလင်းချက်

  • myNumbers.length က Array ရဲ့ length ဖြစ်တယ်။
  • myNumbers.length - 1 က နောက်ဆုံး index ကို ရယူဖို့ပါ။
  • myNumbers[4] က 50 ကို return ပြန်မှာပါ။

နောက်နေ့မှာတော့ Array တွေနဲ့ loops သုံးတာတွေကို သွားမယ်နော်။ 😊
