Day 23: List (Array) ထဲက ဂဏန်းတွေကို ပေါင်းခြင်း (Summing Numbers in a List/Array)

မင်္ဂလာပါ! 👋 လွန်ခဲ့တဲ့ရက်တွေက for, for...of, while စတဲ့ Loop အမျိုးအစားတွေအကြောင်း လေ့လာခဲ့ကြတယ်နော်။ ဒီနေ့မှာတော့ Loop တွေရဲ့ အသုံးဝင်ပုံကို ပြသနိုင်မယ့် လက်တွေ့ကျတဲ့ ဥပမာတစ်ခုကို လုပ်ကြည့်ရအောင်။ အဲ့ဒါကတော့ Array ထဲမှာရှိတဲ့ ဂဏန်းတွေအားလုံးကို ပေါင်းနည်း ပါပဲ 😉။

ဘာကြောင့် Array ထဲက ဂဏန်းတွေကို ပေါင်းဖို့ လိုအပ်တာလဲ?

စဉ်းစားကြည့်ပါ... မင်းမှာ ကျောင်းသားတစ်ယောက်ရဲ့ ဘာသာရပ်အမှတ်စာရင်း (ဥပမာ: [၇၀, ၈၅, ၉၀, ၇၅, ၈၀]) ရှိတယ်ဆိုပါစို့။ အဲ့ဒီအမှတ်တွေရဲ့ စုစုပေါင်းကို တွက်ချင်ရင် တစ်ခုချင်းစီ လက်နဲ့ ပေါင်းနေတာက မလွယ်ပါဘူး။ အထူးသဖြင့် ဂဏန်းတွေ အများကြီးရှိနေတဲ့အခါပေါ့။ ဒီလိုအချိန်မျိုးမှာ Loop ကိုသုံးပြီး အမှတ်အားလုံးကို အလိုအလျောက် ပေါင်းလို့ရပါတယ်။

Loop ကိုသုံးပြီး ဂဏန်းတွေ ပေါင်းနည်း

ဒီလုပ်ဆောင်ချက်အတွက် for...of loop က အရမ်းအဆင်ပြေပါတယ်။ ဘာလို့လဲဆိုတော့ Index ကို စိတ်ပူစရာမလိုဘဲ Array ထဲက ဂဏန်းတန်ဖိုးတစ်ခုချင်းစီကို တိုက်ရိုက် ရယူပြီး ပေါင်းထည့်သွားနိုင်လို့ပါ။

ရိုးရှင်းသော Code ဥပမာ (JavaScript ဖြင့်):

let examScores = [70, 85, 90, 75, 80]; // စာမေးပွဲ အမှတ်စာရင်း Array
let totalScore = 0; // စုစုပေါင်း အမှတ်ကို သိမ်းထားဖို့ Variable (စစချင်း 0 ပေးထားတယ်)

// examScores Array ထဲက အမှတ်တစ်ခုချင်းစီကို လှည့်ပတ်ကြည့်မယ်
for (let score of examScores) {
  totalScore = totalScore + score; // လက်ရှိ score ကို totalScore ထဲ ပေါင်းထည့်မယ်
  // ဒါကို totalScore += score; လို့လည်း ရေးလို့ရပါတယ်
}

console.log("စုစုပေါင်း အမှတ်: " + totalScore); // ရလဒ်ကို ဖော်ပြမယ်

Code ရဲ့ ရှင်းလင်းချက်:

  • let examScores = [70, 85, 90, 75, 80];: ဒါက examScores ဆိုတဲ့ နံပါတ်တွေပါတဲ့ Array တစ်ခုကို ဖန်တီးလိုက်တာပါ။
  • let totalScore = 0;: ဒါက ဂဏန်းတွေပေါင်းပြီးရလာမယ့် စုစုပေါင်းတန်ဖိုးကို သိမ်းထားဖို့ totalScore ဆိုတဲ့ Variable တစ်ခုကို ဖန်တီးလိုက်တာပါ။ စစချင်းမှာ 0 လို့ သတ်မှတ်ထားရပါတယ်။ (ဘာလို့လဲဆိုတော့ ဘာမှ မပေါင်းခင် စုစုပေါင်းက 0 ပဲ ဖြစ်ရမှာမို့ပါ)။
  • for (let score of examScores): ဒါက examScores Array ထဲက အမှတ်တစ်ခုချင်းစီကို လှည့်ပတ်ဖို့ for...of loop ကို စတင်လိုက်တာပါ။ Loop တစ်ပတ်စီမှာ score ဆိုတဲ့ Variable ထဲကို Array ထဲက လက်ရှိ အမှတ်တစ်ခု ရောက်လာပါမယ်။
  • totalScore = totalScore + score;: ဒါက Loop တစ်ပတ်စီမှာ လုပ်ဆောင်မယ့် အဓိက လုပ်ဆောင်ချက်ပါ။ totalScore ရဲ့ လက်ရှိတန်ဖိုးကို score ရဲ့ တန်ဖိုးနဲ့ ပေါင်းပြီး ရလာတဲ့ အဖြေကို totalScore ထဲကိုပဲ ပြန်ထည့်လိုက်တာပါ။

ဒီ Code ကို Run လိုက်ရင် ဘာထွက်လာမလဲ?

စုစုပေါင်း အမှတ်: 400

ဒီနေ့မှာတော့ Loop တွေကို သုံးပြီး Array ထဲက ဂဏန်းတွေကို ဘယ်လို ပေါင်းရမလဲဆိုတာကို လေ့လာခဲ့ကြပါတယ်။ ဒါဟာ Programming မှာ အချက်အလက်တွေကို စုစည်းပြီး တွက်ချက်မှုတွေ လုပ်ဆောင်တဲ့နေရာမှာ မရှိမဖြစ် လိုအပ်တဲ့ Skill တစ်ခုပါပဲ။ ဒီနေ့သင်ခန်းစာကို စမ်းသပ်လေ့ကျင့်ကြည့်ဖို့ တိုက်တွန်းပါတယ်နော်! 😊
