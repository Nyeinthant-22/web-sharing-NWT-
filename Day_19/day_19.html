Day 19: List (Array) ထဲက အချက်အလက်တွေကို ဖယ်ရှားခြင်း

မင်္ဂလာပါ! 👋 မနေ့က Array ထဲကို အသစ်ထည့်နည်းကို လေ့လာခဲ့တယ်နော်။ ဒီနေ့တော့ ဖယ်ရှားနည်းတွေကို လေ့လာမယ်။

1. pop() နည်းလမ်း

pop() က Array ရဲ့ နောက်ဆုံးမှာရှိတဲ့ တန်ဖိုးကို ဖယ်ရှားတယ်။ 

let myFruits = ["apple", "banana", "orange", "grape"];
let removedFruit = myFruits.pop();
console.log(myFruits);      // ["apple", "banana", "orange"]
console.log(removedFruit);  // "grape"

ရှင်းလင်းချက်

  • pop() က နောက်ဆုံး index မှာရှိတဲ့တန်ဖိုးကိုဖယ်ရှားတယ်။
  • ဖယ်ရှားတဲ့တန်ဖိုးကို Variable ထဲသိမ်းလို့ရတယ်။

2. shift() နည်းလမ်း

shift() က အစမှာရှိတဲ့တန်ဖိုးကိုဖယ်ရှားတယ်။

let myNumbers = [10, 20, 30, 40];
let removedNumber = myNumbers.shift();
console.log(myNumbers);      // [20, 30, 40]
console.log(removedNumber);  // 10

ရှင်းလင်းချက်

  • shift() က အစဆုံးတန်ဖိုးကိုဖယ်ရှားတယ်။
  • ဖယ်ရှားတဲ့တန်ဖိုးကိုလည်း သိမ်းလို့ရတယ်။

နောက်တစ်နေ့မှာတော့ Array ထဲက တခုချင်းစီကို ဘယ်လိုဖျက်မလဲဆိုတာ တိုးမြှင့်ပြီး ဆက်လေ့လာမှာပါ။ 😊
