Day 9: အချက်အလက် အမျိုးအစား အခြေခံ (Basic Data Types)

မင်္ဂလာပါ! 👋 မနေ့က Variable တွေအကြောင်း လေ့လာခဲ့ကြတယ်နော်။ Variable တွေဟာ အချက်အလက်တွေကို သိမ်းဆည်းတဲ့ "စက္ကူပုံးလေးတွေ" လိုပဲဆိုတာ မှတ်မိကြမယ်ထင်ပါတယ်။ ဒီနေ့မှာတော့ အဲ့ဒီ "စက္ကူပုံး" တွေထဲမှာ ထည့်နိုင်တဲ့ အချက်အလက် အမျိုးအစား တွေအကြောင်း ပြောပြမယ်။ မတူညီတဲ့ အရာတွေအတွက် မတူညီတဲ့ ပုံးတွေ သုံးသလိုမျိုးပေါ့ 😉။

Programming မှာလည်း သိမ်းဆည်းမယ့် အချက်အလက်ရဲ့ အမျိုးအစားပေါ်မူတည်ပြီး Variable တွေကို အမျိုးအစား ခွဲခြားသတ်မှတ်ထားပါတယ်။ အခု အခြေခံ အသုံးအများဆုံး အမျိုးအစား (၃) ခုကို လေ့လာကြည့်ရအောင် 👇

၁။ နံပါတ်များ (Numbers)

ဒါကတော့ ကျွန်တော်တို့ အမြဲသုံးနေတဲ့ ဂဏန်းတွေပါပဲ! Programming မှာ နံပါတ်တွေကို အဓိကအားဖြင့် နှစ်မျိုးခွဲခြားနိုင်တယ်:

  • Integer (ပြည့်ကိန်း): ဒဿမ မပါတဲ့ နံပါတ်တွေ (ဥပမာ: -၅, ၀, ၁၀, ၁၀၀၀)
  • Float (ဒဿမကိန်း): ဒဿမ ပါတဲ့ နံပါတ်တွေ (ဥပမာ: -၃.၁၄, ၀.၅, ၉.၈၆)

အရေအတွက် = ၅ (ပြည့်ကိန်း Variable)

အပူချိန် = ၂၆.၅ (ဒဿမကိန်း Variable)

၂။ စာသားများ (Text / Strings)

စာလုံးတွေ၊ စကားလုံးတွေ၊ စာကြောင်းတွေကို သိမ်းဖို့ သုံးတဲ့ အမျိုးအစားပါ။ Programming မှာ စာသားတွေကို "" (quote) နှစ်ခုကြားမှာ ရိုက်ပါတယ်။

နာမည် = "ခင်ခင်" (စာသား Variable)

နှုတ်ခွန်းဆက် = "မင်္ဂလာပါ!" (စာသား Variable)

၃။ မှန်/မှား တန်ဖိုးများ (Booleans)

တန်ဖိုး နှစ်ခုပဲ ရှိပါတယ်: True နဲ့ False။ "ဟုတ်"/"မဟုတ်" အဖြစ် နားလည်လို့ရပါတယ်။

မိုးရွာနေလား = ဟုတ် (Boolean Variable)

အသက် ၁၈ နှစ် ပြည့်ပြီလား = မဟုတ် (Boolean Variable)

ဒီနေ့မှာတော့ အခြေခံ အချက်အလက် အမျိုးအစား (၃) ခုကိုပဲ သိထားဖို့ လုံလောက်ပါပြီ။ နောက်ပိုင်းမှာတော့ ဒီအမျိုးအစားတွေနဲ့ ပတ်သက်တဲ့ လုပ်ဆောင်ချက်တွေကို ဆက်လက် လေ့လာသွားမှာဖြစ်ပါတယ်။ 😊
