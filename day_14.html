Day 14: တစ်ပတ်အတွင်း လေ့လာခဲ့တာတွေကို ပြန်ကြည့်ရအောင်

မင်္ဂလာပါ အားလုံးပဲ! 👋 ဒါနဲ့ဆို Programming အခြေခံ ဒုတိယအပတ်လည်း ပြီးဆုံးသွားပြီနော် 😊။ ဒီတစ်ပတ်မှာ သင်ယူခဲ့တဲ့ အဓိကအကြောင်းအရာတွေကို အတူပြန်ကြည့်ကြမယ်။

Day 8: Variable ဆိုတာ ဘာလဲ?

Variable ဆိုတာ အချက်အလက်တွေကို သိမ်းဆည်းဖို့ သုံးတဲ့ "နာမည်" ပါတဲ့ "စက္ကူပုံးလေးတွေ" နဲ့ တူတယ်ဆိုတာကို လေ့လာခဲ့တယ်။

Day 9: အချက်အလက် အမျိုးအစား အခြေခံ

နံပါတ် (ints & floats), စာသား (strings), မှန်/မှား (booleans) တို့နဲ့ မိတ်ဆက်ခဲ့တယ်။

Day 10: သင်္ချာ လုပ်ဆောင်ချက်တွေ

ပေါင်းခြင်း (+), နုတ်ခြင်း (-), မြှောက်ခြင်း (*), စားခြင်း (/) တို့နဲ့ တွက်ချက်နိုင်တာကို လေ့လာခဲ့တယ်။

Day 11: နှိုင်းယှဉ်ခြင်း လုပ်ဆောင်ချက်

==, !=, >, <, >=, <= တို့ကို သုံးပြီး တန်ဖိုးနှိုင်းယှဉ်နည်းကို သင်ယူခဲ့တယ်။

Day 12: အခြေအနေအပေါ် မူတည်တဲ့ လုပ်ဆောင်ချက် - If

အခြေအနေတစ်ခုကို စစ်ပြီး မှန်ရင်လုပ်တာမျိုးကို "If" နဲ့ ပြုလုပ်တတ်လာခဲ့တယ်။

Day 13: If နဲ့ ပတ်သက်တဲ့ ရိုးရှင်းသော ဥပမာများ

နေ့စဉ်ဘဝ ဥပမာတွေနဲ့ "If" ကို ပိုမို နားလည်လာစေခဲ့တယ်။

ဒီတစ်ပတ်မှာ သင်ယူခဲ့တာတွေက Programming ရဲ့ အခြေခံ အုတ်မြစ်တွေပါ။ နားလည်ပြီးသားဆိုရင် နောက်လေ့လာမယ့် concept တွေ ပိုမိုလွယ်ကူမှာပါ။

မေးစရာ ရှိရင် comment ထဲမှာ မေးဖို့ လွတ်လပ်ပါတယ် 🤔။

နောက်တစ်ပတ်မှာတော့ ဒီအခြေခံတွေကို ပေါင်းပြီး ပို စိတ်ဝင်စားဖို့ကောင်းတဲ့ နည်းပညာတွေကို ဖန်တီးဖို့ စတင်ပါမယ်။ ဆက်လက် လေ့လာကြပါဦး။ 👋
