Day 15: List ဆိုတာ ဘာလဲ?

မင်္ဂလာပါ! ဒီနေ့မှာတော့ List (Array) ဆိုတာ ဘာလဲဆိုတာ လေ့လာသွားမှာ ဖြစ်ပါတယ်။

List (Array) ဆိုတာ ဘာအတွက်သုံးတာလဲ?

Variable တစ်ခုချင်းစီနဲ့ သိမ်းတာထက် စုစည်းပြီး သိမ်းထားလို့ ပိုမိုလွယ်ကူပါတယ်။ 

let friend1 = "မောင်မောင်";
let friend2 = "ခင်ခင်";
let friend3 = "ကိုကို";
let friend4 = "အေးအေး";

နေ့စဉ်ဘဝနဲ့ ဥပမာ

  • သူငယ်ချင်းစာရင်း: ["မောင်မောင်", "ခင်ခင်", "ကိုကို", "အေးအေး"]
  • ဝယ်စရာ: ["ဆန်", "ဆီ", "ကြက်ဥ", "ဟင်းသီးဟင်းရွက်"]
  • အမှတ်: [၈၅, ၉၂, ၇၈, ၉၅]

JavaScript Code Example

let myFriends = ["မောင်မောင်", "ခင်ခင်", "ကိုကို", "အေးအေး"];

console.log(myFriends[0]);  // "မောင်မောင်"
console.log(myFriends[2]);  // "ကိုကို"

Code ရဲ့ ရှင်းလင်းချက်

  • let myFriends = [...]: Array တစ်ခုဖန်တီးခြင်း
  • myFriends[0]: ပထမ နာမည်ကို ရယူခြင်း
  • myFriends[2]: တတိယ နာမည်ကို ရယူခြင်း

Array ဆိုတာ ဘာလဲ?

JavaScript မှာ array ဆိုတာ တန်ဖိုးများစွာကို တစ်စုတည်း သိမ်းဆည်းနိုင်တဲ့ data structure တစ်ခုဖြစ်ပါတယ်။

ဆိုတော့ နောက်ရက်တွေမှာ List (Array) တွေကို Programming မှာ ဘယ်လို ဖန်တီးမလဲ၊ List ထဲက အချက်အလက်တွေကို ဘယ်လို ထုတ်ယူမလဲ၊ ပြီးတော့ List (Array) တွေနဲ့ ဘာတွေ လုပ်လို့ရလဲဆိုတာတွေကို ပိုပြီး အသေးစိတ် လေ့လာသွားကြမယ်နော်! 😊
